Komponentor mini demo

What’s going on?

index.html loads jQuery, Komponentor, and KSimpleViews, then calls komponentor.root("#app", "app.html").

Files

  • app.html — shell: shared theme model, hash router on #view, opens the modal intent
  • list.html / item.html — routed into #view (#/ and #/item/:id)
  • about.html — headless intent (modal); receives the same KModel via .data("theme", …)

Try this

  1. Click List / an item link — watch #view swap HTML by route.
  2. Open Theme (intent) — modal loads without replacing the shell.
  3. Change the color — preview above the nav and swatch in the modal stay in sync (KModel + KView).
